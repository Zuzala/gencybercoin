GenCyberCoin website has a bug bounty program! This program will reward you with coins if you find bugs (problems, errors) on the website.

Try to do something that the website would not expect a normal user to do. Afterwards, you may see a message that you found a bug and you will automatically receive a reward for that! For instance, if you submit something on the website, the information will be processed by the server and the result will be shown to you on the page. What if you send something that a normal user would never think to send? Maybe you find a place where you could request a certain type of information from the server or you can try to tinker around on the page itself.

Bug Bounty hunting is a real thing and people receive monetary $$ rewards on such websites as HackerOne and BugCrowd. It can be quite profitable to participate in these "ethical hacking" But Bounty programs. For example, a 10-year-old was rewarded $10,000 by Facebook in 2016 for finding a serious bug on Instagram.

Also, read a short essay from a famous Bruce Schneier on "good VS bad hackers" and if you are interested in checking out some real top-level security investigations, drop by KrebsOnSecurity.

What happens if someone performed an attack in an unethical way? They may go to prison and owe a large fine as the MalwareTech's case. So, make sure that you always, ALWAYS think about ethics first.

For this website though, you can try to break it in any way you want. A helpful hint: you can always right-click on a specific item on any page and select Inspect to see the underlying code.
